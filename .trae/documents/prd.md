## 1. 产品概述

面向连锁口腔诊所运营经理的器械包备包效率与周转风险数据看板。通过实时监控各门店备包情况、分析包型利用率异常、记录缺包复盘事件，帮助管理层用少量数据快速发现备包瓶颈，降低午高峰断包率，优化器械包配置与排程。

- 核心问题：门店午高峰频繁断包、包型配置不均衡、缺包原因缺乏追踪
- 目标用户：连锁口腔诊所运营经理、护士长
- 价值：减少缺包事件30%+，优化包型配比，形成闭环复盘机制

## 2. 核心功能

### 2.1 用户角色

| 角色 | 核心权限 |
|------|----------|
| 运营经理 | 查看所有门店看板数据、包型分析排行、周报简报 |
| 护士长 | 填写缺包复盘记录、查看本门店数据 |

### 2.2 功能模块

1. **门店备包总览页**：按门店展示今日预约量、实际备包数、缺包次数、加急次数，支持午高峰断包高亮预警
2. **包型利用率分析页**：包型利用率图表、异常门店/时段/器械排行榜
3. **复盘记录与周报页**：缺包事件复盘表单、周度简报自动生成与增配建议

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 门店备包总览 | 门店指标卡片组 | 按门店展示今日预约量、实际备包数、缺包次数、临时改项目加急次数，断包门店红色高亮 |
| 门店备包总览 | 午高峰预警面板 | 标注当前午高峰时段（11:00-14:00）缺包门店，突出显示连续断包门店 |

| 包型利用率分析 | 包型利用率总览 | 各包型利用率柱状图（洁牙包、根管包、外科包、正畸包、修复包等），标注闲置/紧张状态 |
| 包型利用率分析 | 异常门店排行 | 缺包次数TOP5门店排行，含缺包率指标 |

| 复盘记录与周报 | 复盘填写表单 | 护士长填写缺包原因：预约变更、消毒锅排程、器械损坏、人员漏备，支持备注补充 |
| 复盘记录与周报 | 周度简报 | 自动汇总本周缺包数据，给出增配包、调整备包时间、培训新护士的建议 |

## 3. 核心流程

**运营经理日常巡检流程**：登录看板 → 查看门店备包总览 → 发现断包门店 → 进入包型分析查看异常排行 → 查看复盘记录确认原因 → 依据周报建议调整配置

**护士长复盘流程**：发生缺包事件 → 进入复盘记录页 → 填写缺包原因与备注 → 提交复盘 → 系统纳入周报统计

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深青蓝 (#0F2B3C) 背景搭配明亮的青绿色 (#00D4AA) 作为强调色，传递医疗专业感与数据活力
- 辅助色：警告红 (#FF4757) 用于缺包预警，琥珀色 (#FFA502) 用于紧张提示，灰色系用于中性数据
- 按钮：圆角8px，填充式主按钮为青绿色，危险操作为红色
- 字体：数据区使用等宽风格字体 (JetBrains Mono)，标题区使用思源黑体风格
- 布局：左侧固定导航栏 + 右侧内容区，卡片式模块布局
- 图标：线性风格，2px描边，与数据看板专业调性匹配

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 门店备包总览 | 门店指标卡片组 | 横向滚动卡片，每卡片含门店名、4项指标数字、断包红标动画 |
| 门店备包总览 | 午高峰预警面板 | 顶部横幅式面板，脉冲动画红点标记断包门店 |
| 门店备包总览 | 门店对比趋势 | 多折线图，青绿主色+辅助色区分门店，悬浮tooltip展示详情 |
| 包型利用率分析 | 包型利用率总览 | 水平柱状图，绿色(充裕)/黄色(适中)/红色(紧张)渐变色带 |
| 包型利用率分析 | 异常门店排行 | 竖向排行榜卡片，TOP3用金银铜色标识 |
| 包型利用率分析 | 异常时段排行 | 热力图样式，颜色深浅表示缺包频率 |
| 包型利用率分析 | 常缺器械排行 | 横向条形图，红色渐变从浅到深表示频率 |
| 复盘记录与周报 | 复盘记录列表 | 表格+筛选器，原因标签彩色pill样式 |
| 复盘记录与周报 | 复盘填写表单 | 侧滑抽屉式表单，4个原因单选+备注多行文本 |
| 复盘记录与周报 | 周度简报 | 简报卡片组，含数据摘要+建议列表，支持导出 |

### 4.3 响应式

桌面优先设计，最小适配1280px宽度。左侧导航栏可折叠，内容区网格自适应。移动端仅保留核心数据卡片和预警信息。
